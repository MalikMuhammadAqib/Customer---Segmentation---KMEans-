# Customer Segmentation Using KMeans

## Business Problem
A retail mall wants to segment customers for targeted marketing.

## Objective
Use clustering to identify meaningful customer segments.

## Tools Used
- Python
- Scikit-learn
- Pandas
- Seaborn
- Google Colab

## Methodology
1. Data Cleaning
2. EDA
3. Feature Scaling
4. Elbow Method
5. Silhouette Score
6. KMeans Clustering

## Results
- Optimal clusters: 5
- Clear customer segmentation identified

## Business Insights
- Premium Customers
- Impulsive Buyers
- Conservative Wealthy
- Budget Customers
